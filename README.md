## Description

RU:
  Програма представляет собой простой кликер на tkinter, при нажатии на кнопку **'Click Me!'** счет увеличивается на **1**\
  При нажатии на кнопку **'Delete score' ВНИМАНИЕ** счет **обнулится**!\
ENG:
  The program is a simple tkinter clicker. When you click the **'Click Me!'** button, the score increases by **1**\
  When you click the **'Delete score' WARNING** button, the score **resets**!
